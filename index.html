    About Group Capstone JS

    This is a JavaScript capstone project. It is a web application based on external APIs. It retrieves information from two different APIs: Meals DB and Involvement API.
    The homepage lists some recipes that we can "like".
    The comment button opens a popup window showing more information about the recipe and we can use it to write a comment.
